BaatCheet is a full-stack real-time chat application built using the MERN stack and Socket.io. It enables users to communicate instantly with secure authentication, online user tracking, and a responsive user interface.

The application supports real-time text and image messaging, user profile management, theme switching.

Tech Stack:
- Frontend: React, Tailwind CSS, DaisyUI
- Backend: Node.js, Express, MongoDB
- Real-Time Communication: Socket.io
- Authentication: JWT
- Deployment: Render
